Common Issues & Solutions
Login Issues
  • Forgot Password? Use the "Forgot Password?" link on the login page to reset via email OTP
  • Can't Login? Try OTP login method using your registered email
  • Account Locked? Contact administrator for account assistance
OTP Issues
  • OTP Not Received? Check spam/junk folder, verify email address
  • OTP Expired? Request a new OTP, codes expire in 10 minutes
  • Invalid OTP? Ensure you're entering the latest 6-digit code
Bot Issues
  • Bot Not Responding? Check bot activation status in admin dashboard
  • Bot Token Error? Verify TELEGRAM_BOT_TOKEN in environment
  • Bot Commands? Use /help command in Telegram for available commands
